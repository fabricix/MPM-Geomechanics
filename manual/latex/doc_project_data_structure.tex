\section{Project data structure}\label{sec:project-data-structure}

The directories with files and folders in this project are organized as follows:

\begin{itemize}
	\item \texttt{src/ build/ qa/ external/ and inc/} sources, builds qa(tests and benchmarking), google-test folder and C++ headers files
	\item \texttt{manual/ docs/ } program manual and documentation for doxygen
	\item \texttt{tests/} simulation results of analytical and numerical tests
	\item \texttt{examples/} application examples
\end{itemize}
