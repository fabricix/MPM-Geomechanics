\subsection{Load and save states}

In static analysis, the quasi static analysis can be store in to a simulation state:

\begin{lstlisting}[language={},caption={input JSON}]
	"save_state":true,
\end{lstlisting}

And for loading the salved state

\begin{lstlisting}[language={},caption={input JSON}]
	"load_state":true,
\end{lstlisting}
